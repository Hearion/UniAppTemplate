# UniAppTemplate

## 项目启动

- 需要 Node.js 版本 `15.1.0`

```
yarn install
```

## 集成方法

- 文件转 base64
- 轻提示
- 创建图片压缩至2M
- 获取集合中，对应的属性的值
- 常用正则
- 可参考 [PrjUtils.js](src%2Futils%2FPrjUtils.js)
